의도 분석 결과: {intent_result}

위 정보를 바탕으로 실행 계획을 수립하세요.

다음 형식의 JSON으로 응답하세요:
```json
{
    "strategy": "direct" | "sequential" | "parallel",
    "execution_plan": [
        {
            "step": 1,
            "executor": "search" | "analysis" | "document",
            "action": "구체적인 작업 설명",
            "expected_output": "예상 결과물"
        }
    ],
    "reasoning": "전략 선택 근거"
}
```

실행 전략 설명:
- direct: 단일 Executor로 즉시 실행
- sequential: 여러 Executor를 순차 실행 (선행 결과를 후행이 활용)
- parallel: 여러 Executor를 병렬 실행 (독립적인 작업들)

Executor 설명:
- search: 검색 실행 (법률, 부동산, 대출 검색)
- analysis: 분석 실행 (데이터 분석, 인사이트 도출)
- document: 문서 실행 (보고서, 계약서 등 생성)
